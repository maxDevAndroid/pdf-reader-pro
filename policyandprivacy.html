Política de Privacidade - PDF Reader Pro

Última atualização: 18/05/2026

1. Introdução

Bem-vindo à Política de Privacidade do aplicativo PDF Reader Pro, desenvolvido pela empresa TriVisionTech. Esta página explica como coletamos, usamos e protegemos as informações dos usuários do aplicativo.

2. Coleta de Dados

O aplicativo PDF Reader Pro não coleta dados pessoais identificáveis. Todos os arquivos PDF abertos no aplicativo permanecem apenas no dispositivo do usuário.

3. Permissões do Dispositivo

Para funcionar corretamente, o aplicativo pode solicitar permissões como:

  • Acesso ao armazenamento do dispositivo para abrir arquivos PDF;
  • Leitura de arquivos locais;
  • Compartilhamento de arquivos, caso o usuário utilize essa função.

Nenhum arquivo é enviado para servidores externos.

4. Publicidade

O aplicativo PDF Reader Pro pode exibir anúncios fornecidos por parceiros de publicidade. Esses anúncios podem coletar dados anônimos, como identificadores de dispositivo, apenas para fins de personalização e análise de desempenho. Para mais informações, consulte a política de privacidade dos provedores de anúncios.

5. Uso de Dados

Os dados coletados por parceiros de anúncios são anônimos e não identificam pessoalmente o usuário. TriVisionTech não vende, compartilha ou distribui qualquer informação pessoal a terceiros.

6. Segurança

Todos os arquivos manipulados pelo aplicativo permanecem localmente no dispositivo do usuário. O aplicativo não possui servidores próprios nem realiza upload de documentos.

7. Alterações nesta Política

A empresa TriVisionTech pode atualizar esta Política de Privacidade periodicamente. Recomendamos que os usuários revisem esta página regularmente.
